CSS example

In this example, we condense three rules with identical declaration into one. Thus,

  • h1 { font-family: sans-serif }
  • h2 { font-family: sans-serif }
  • h3 { font-family: sans-serif }

is equivalent to:

h1, h2, h3 { font-family: sans-serif }
